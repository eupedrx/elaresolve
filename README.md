# 💡 Ela Resolve

[🔗 Acesse o site aqui](https://elaresolve.vercel.app/)

**Ela Resolve** é um site desenvolvido com foco em **soluções práticas e acessíveis para o dia a dia**, destacando serviços, conteúdos e iniciativas voltadas principalmente ao público feminino. O projeto foi desenvolvido com uma abordagem moderna, responsiva e otimizada para performance e experiência do usuário.

## 🛠️ Tecnologias Utilizadas

O site foi construído com as seguintes tecnologias:

- **Next.js** – Framework React moderno com foco em performance, SSR e otimização.
- **React.js** – Biblioteca para criação de interfaces de usuário reativas e componentizadas.
- **Tailwind CSS** – Framework de utilitários CSS para criação de layouts rápidos e responsivos.
- **TypeScript** – Superset do JavaScript que adiciona tipagem estática e maior segurança no desenvolvimento.
- **Vercel** – Plataforma de deploy e hospedagem otimizada para aplicações Next.js.

## 🚀 Funcionalidades

- Layout 100% responsivo e adaptado para dispositivos móveis.
- Estrutura de componentes reutilizáveis.
- SEO otimizado.
- Navegação rápida e suave (Client-side routing).
- Performance otimizada para Core Web Vitals.

## 📦 Como rodar o projeto localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/elaresolve.git
   cd elaresolve
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Rode o projeto localmente:
   ```bash
   npm run dev
   ```

4. Acesse: `http://localhost:3000`

## 📁 Estrutura do Projeto

- `/components` – Componentes reutilizáveis da interface.
- `/pages` – Rotas principais da aplicação.
- `/public` – Arquivos estáticos como imagens e ícones.
- `/styles` – Configurações e estilos globais com Tailwind.
- `/utils` – Funções auxiliares, se houver.
